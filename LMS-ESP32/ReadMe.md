# LMS ESP32 Chassis

Das Gehäuse für den LMS von Antons Mindstorms (https://www.antonsmindstorms.com/) schützt die kleine Platine vor unvorsichtiger Handhabung optimal.

Das Gehäuse hat dabei Aussparungen für die Stepper-Motoranschlüße, GPIO Ports, die Grove-Ports und den Display-Anschluss, sowie "freie Sicht" auf die 2 LEDs.
Um das Gehäuse fest mit dem ESP zu vereinen, kann man 2-4 "Doppelstöpsel" nutzen.
